Project: Asynchronous transaction log agrregation system - ATLAS
Description: Atlas is a distributed event-driven processing pipeline system that ingests high volume of real-time data through Kafka, performs streaming transformations and anomaly detection.

The data is stored in PostgresSQL for low latency analytics and these analytics are charted using Grafana

Pre-requisites:
1. Zookeeper
2. Kafka
3. PostgresSQL
